폼 관련 태그

        action - 폼의 입력된 값들을 전송받을 서버의 클래스명
        method - get 또는 post 방식으로 서버전송 방식을 지정
            - get : url 에 넘어가는 데이터가 보임
            - post : url 에 넘어가는 데이터가 보이지 않음
